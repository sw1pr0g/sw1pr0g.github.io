## Development

Make sure you have `go` (also configured paths for packages) and `air` installed.

1. https://go.dev/doc/install
2. `go install github.com/air-verse/air@latest`

Then run:

1. `make develop`
2. Done! All hosted on [localhost:8080](http://localhost:8080)
